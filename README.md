# IES_Alarcos_Mbot
Repositorio para la competición de robótica de la UCLM en la modalidad Mbot2.

En la carpeta "Code" aparece el código de Mblock para las tres misiones.

## Características del robot
Se emplea la configuración básica de Mbot2 con sensor de ultrasonidos y el cuádruple sensor de color RGB para las misiones 1 y 2, mientras que para la misión 3 se añade en serie con el sensor de ultrasonidos, el sensor de llama.

Las modificaciones sobre el montaje es el desplazamiento hacia la izquierda del cuádruple sensor de color RGB, para poder seguir las líneas por la izquierda. Requisito en la misión 1 que penaliza estar tocando dos zonas de fumigación.
